Xcel Energy Requirements

  • File Solar*Rewards application — system ≤20kW DC qualifies
  • Net metering interconnection agreement required before PTO
  • System cannot exceed 120% of historical usage at ideal conditions
  • UL1741-SA certified inverters required (FlexBOSS21 ✓)
  • Anti-islanding protection required in all grid-tied inverters
  • Utility-visible lockable disconnect — confirm if required by Xcel
  • Production meter may be required — confirm with Xcel
  • Xcel inspection required before Permission to Operate (PTO)
  • Solar*Rewards: $0.03/kWh × all production × 10 years
  • Battery incentive: $175/kWh upfront, capped at $5,000

NEC 2023 Key Code References

  • 690.12 — Rapid shutdown: all DC circuits >1.5V/1A inside array boundary
  • 690.15 — DC source disconnect required at each array location
  • 690.47 — All PV module frames bonded to equipment grounding conductor
  • 250.24 — Single neutral-ground bond at service entrance (GridBOSS)
  • 250.32 — Separate building subpanel rules — N-G bond removed at interior panel
  • 705 — Interconnected power production sources (solar + grid)
  • 706 — Energy storage systems — battery installation, location, protection
  • 230 — Service entrance, overcurrent protection, disconnecting means
  • All conductors in conduit or listed raceway per NEC 300
  • Non-combustible mounting surface for inverters and batteries

Installation Sequence

  • 1. Pull permits: City of Bloomington building + electrical
  • 2. File Xcel Solar*Rewards + interconnection application
  • 3. Coordinate Xcel 200A service lateral upgrade (Xcel work)
  • 4. Install sonotube footings; set J-bolts; cure concrete (7 days min)
  • 5. Erect pergola frames; install rafters; torque all hardware
  • 6. Mount IronRidge rails; hang panels; complete DC wiring in conduit
  • 7. Install GridBOSS at service entrance; install batteries + inverters
  • 8. Run all AC conductors in conduit; swap interior panel; commission
  • 9. City electrical inspection → Xcel final inspection → PTO
  • 10. Register systems: EG4 warranty + Xcel Solar*Rewards activation
