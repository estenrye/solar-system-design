Inverter Strategy for Dual Array
✓ Recommended
2× EG4 18kPV — Parallel Configuration
One inverter per array, wired in parallel to the same battery bank and service panel. Each 18kPV has 3 MPPTs — dedicated string management per array maximizes harvest. Combined output: 24kW continuous. Clean, redundant — if one inverter fails, the other keeps running.
Per inverter DC input6.02 kW (2 strings of 7)
Combined AC output24 kW continuous
MPPTs used per unit2 of 3 (1 spare for expansion)
Cost (×2)~$5,600
Redundancy✓ Full — independent units
Alternative
1× EG4 18kPV + EG4 GridBOSS
Single inverter handles both arrays via the GridBOSS distribution panel. The 18kPV has 3 MPPTs total — you'd use all 3, splitting the 28 panels across strings. Simpler wiring, lower cost, but no redundancy and the 18kPV's 18kW PV input limit is approached at 12 kW DC (comfortable margin exists).
DC input to single 18kPV12.04 kW (within 18kW limit)
